# Images Directory

Place the following images in this directory:

## Required Images:

1. **wardrobe-icon.png** (80x80px)
   - Logo/icon for auth pages
   - Suggested: Clothes hanger icon or wardrobe illustration

2. **wardrobe-illustration.png** (400x400px)
   - Welcome banner illustration
   - Suggested: Colorful wardrobe or clothing rack illustration

3. **no-items-placeholder.png** (300x300px)
   - Empty state illustration
   - Suggested: Empty wardrobe or "add items" illustration

4. **outfit-placeholder.png** (200x200px)
   - Default outfit image
   - Suggested: Outfit mannequin or clothing combination

## Video:

Place background video in:
- **public/media/background-video.mp4**
- **public/media/background-video.webm** (optional, for better browser support)

Video should be:
- Abstract/fashion themed
- Loopable
- 10-30 seconds
- 1920x1080 or similar HD resolution
- Compressed (< 10MB)
